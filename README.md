## 🚀 Projects Featured

### 🦷 BrightSmile Dental — AI-Powered Clinic Management System *(Latest)*
A fully automated dental clinic management system featuring an AI receptionist (Aria), real-time appointment booking, EHR integration, and HIPAA-ready architecture.
- **Stack:** n8n · Google Gemini · Cal.com · OpenEMR · Railway · PostgreSQL · Gmail API · Google Calendar API
- **Highlights:** 3 workflows · 6 integrated services · end-to-end booking in under 10 seconds

### 🌿 GlowLab — Automated Ecommerce Marketing System
Complete automated marketing and CRM system for a skincare ecommerce brand. From lead capture to post-purchase follow-up — zero manual intervention after setup.
- **Stack:** GoHighLevel · Shopify · LeadConnector
- **Highlights:** 3 automated workflows · welcome email in under 60 seconds · abandoned cart recovery · 5-stage visual pipeline

### 📊 AI Weekly Sales Report Bot
Automatically pulls weekly sales data from Google Sheets, generates a plain-English AI summary, and delivers it to the store owner's inbox every Monday at 8am.
- **Stack:** n8n · Google Sheets · Groq AI · Gmail

### 💆 Glow Skin Derma Booking Bot
AI-powered Telegram bot "Sofia" that handles end-to-end appointment booking for a skin clinic — 24/7 with no human involvement.
- **Stack:** n8n · Telegram · Groq AI

### 🏠 Real Estate Leads Automation
Captures real estate leads, qualifies them with AI, and instantly notifies agents — response in seconds, not hours.
- **Stack:** Make.com · Groq AI · Gmail

## 🌐 Hosting on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://mikel0816.github.io/Micheal-Rivas-Portfolio`
